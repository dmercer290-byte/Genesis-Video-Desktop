# Genesis Video Desktop — Fork Notice

This project is a fork of [LTX Desktop](https://github.com/Lightricks/LTX-Desktop) by Lightricks Ltd.

## Changes Made by Genesis Studio

- Added universal model registry (`models.genesis.json`)
- Added remote backend mode (server-side generation, PC as UI only)
- Added Genesis API shim for multi-engine support (LTX, Wan, HunyuanVideo, AnimateDiff, ComfyUI)
- Added UI panels: model picker, VAE picker, LoRA picker, workflow picker, GPU/server status
- Backend URL, auth token, and server mode now configurable from UI (not env-only)
